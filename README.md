# India AQI Digital Twin v6.0

Real-time Air Quality Index prediction and Digital Twin simulation for 30 Indian cities.

## Models: CNN · LSTM · GRU · GNN
## Simulation: SimPy · SUMO
## XAI: SHAP · LIME

## Quick Start
pip install -r requirements.txt
python src/run_website.py
